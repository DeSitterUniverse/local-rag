# Next.js Framework Guide

Next.js is a React framework that gives you building blocks to create web applications. By framework, we mean Next.js handles the tooling and configuration needed for React.

## Routing
Next.js utilizes a file-system based router built on the concept of pages. When a file is added to the pages directory, it's automatically available as a route.

## Data Fetching
Next.js supports different forms of pre-rendering, including Static Site Generation (SSG) and Server-Side Rendering (SSR), allowing you to pick the best strategy for each page.
